- # 프로젝트

  아웃소싱 프로젝트

  프로젝트 이름 : 헬스 마켓

  프로젝트 내용 : 피트니스 시대에 맞춰 사용자가 원하는 헬스 전용 물품들을 사고 팔 수 있는 중고 거래 사이트


  기능 소개 요약: 파이어베이스를 활용하여 드라마 추천 페이지를 구성하였습니다.

  프로젝트 목표 : 핵심 기능 이해 및 마감시간 전 완성.
